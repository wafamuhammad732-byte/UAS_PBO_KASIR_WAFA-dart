## Fitur Program
Aplikasi CLI ini dirancang untuk memudahkan manajemen pemesanan pada Cafe dengan fitur:
1. **Tambah Pesanan:** Input pesanan dengan rincian khusus (level pedas untuk makanan, ukuran untuk minuman) dilengkapi dengan validasi *input*.
2. **Lihat Pesanan & Total:** Menampilkan seluruh daftar pesanan yang sedang berlangsung beserta perhitungan total biayanya.
3. **Cari Pesanan:** Mencari pesanan secara spesifik berdasarkan nama produk.
4. **Simpan Transaksi:** Simulasi asinkron penyimpanan data pesanan ke dalam *database*.

## Cara Menjalankan Program
1. Pastikan Dart SDK sudah terinstal di komputer.
2. Buka terminal atau *command prompt*.
3. Masuk ke dalam direktori proyek ini (folder `project_uas`).
4. Jalankan perintah berikut di terminal:
   `dart run bin/main.dart`
5. Ikuti instruksi interaktif yang muncul pada layar.
